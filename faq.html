Frequently Asked Questions

  1. Which tld version / tag specification version does the plugin supports?
  2. Does the plugin support more than one tld for each project?
Which tld version / tag specification version does the plugin supports?

At the moment the plugin supports the tld 1.1, 1.2, 2.0 and 2.1 versions.

Does the plugin support more than one tld for each project?

Yes, it does: the tagreference/tag validation reports will list multiple taglib in a single page, tlddoc documentation will be generated for all the taglibraries.
